# FormSelect

Dropdown select component with label, validation, and keyboard navigation.

## Import

```tsx
import { FormSelect } from '@feframework/ui';
```

## Basic Usage

```tsx
<FormSelect
  label="Country"
  options={[
    { value: 'ie', label: 'Ireland' },
    { value: 'ni', label: 'Northern Ireland' },
  ]}
/>
```
